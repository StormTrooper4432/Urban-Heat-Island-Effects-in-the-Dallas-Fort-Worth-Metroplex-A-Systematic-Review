# Website Preview

Run from project root:

```bash
python -m http.server 8000
```

Open:

- http://localhost:8000/website/index.html
- http://localhost:8000/website/methodology.html
- http://localhost:8000/website/results.html
- http://localhost:8000/website/panel-qa.html

Notes:

- Results page reads live CSV files from `outputs/tables`.
- Figure panels render images from `outputs/figures`.
